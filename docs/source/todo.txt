Style guide:

- Headers should be sentence case.
- Use unordered lists or paragraphs for descriptions
- Use ordered lists only for lists of steps
- Use bold for page names, e.g. **Home**.
- Use [] for clickable buttons like **[VISIT]**


- Copy editing
    - Add captions to images (from :alt:)
    - Check sizes on images
    - Make capitalization consistent
    - Make numbers vs. bullet points consistent
    - Fix pdf formatting issues with index, chapters.
